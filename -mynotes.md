

| Feature | SQL (Relational) | NoSQL (Non-Relational) | Vector Database |
| :--- | :--- | :--- | :--- |
| **Data Format** | Structured (Tables, Rows) | Semi-structured (JSON, Key-Value) | Unstructured (Embeddings/Vectors) |
| **Schema** | **Rigid:** Fixed columns/types | **Flexible:** Dynamic fields | **Specific:** High-dimensional arrays |
| **Search Type** | **Exact Match:** SQL Queries | **Pattern Match:** Key-Value lookups | **Similarity Search:** Nearest Neighbor |
| **Scaling** | Vertical (Scale Up) | Horizontal (Scale Out) | Horizontal (Scale Out) |
| **Consistency** | Strong (ACID) | Flexible (BASE/Eventual) | High-speed Retrieval |
| **Primary Goal** | Data Integrity | Scalability & Flexibility | Semantic Understanding/AI |
