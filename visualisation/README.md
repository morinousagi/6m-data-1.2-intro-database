# Normalization Interactive Tutorial

This folder contains an interactive visualization to help learners understand database normalization concepts.

## How to Use

### Opening the Visualization

1. **Option 1 - Double-click**: Simply double-click on `normalization-interactive.html` to open it in your default web browser.

2. **Option 2 - Right-click**: Right-click on the file and select "Open with" > your preferred browser (Chrome, Firefox, Safari, Edge).

3. **Option 3 - Drag and drop**: Drag the HTML file into an open browser window.

### Navigating the Tutorial

- Use the **Next** and **Previous** buttons at the bottom to move through the 8 steps
- You can also use **keyboard arrows** (Left/Right) or **Spacebar** to navigate
- The progress bar at the top shows your current position

### What You'll Learn

The tutorial walks through normalization step-by-step:

| Step | Topic | Description |
|------|-------|-------------|
| 1 | Introduction | The Update Anomaly problem - why normalization matters |
| 2 | Overview | The Three Normal Forms (1NF, 2NF, 3NF) |
| 3 | 1NF Problem | Identifying missing primary keys |
| 4 | 1NF Solution | Adding composite keys |
| 5 | 2NF Problem | Identifying partial dependencies |
| 6 | 2NF Solution | Splitting tables to remove partial dependencies |
| 7 | 3NF Problem | Identifying transitive dependencies |
| 8 | 3NF Solution | Final normalized schema |

### Legend

The visualization uses color coding:
- **Blue** - Primary Key
- **Purple** - Foreign Key
- **Orange/Yellow** - Problem areas (redundancy, dependencies)
- **Green** - Successfully normalized

### Tips for Learners

- Go through each step carefully before clicking Next
- Pay attention to the highlighted cells - they show the problems being addressed
- After completing the tutorial, try the normalization workshop in the lesson
